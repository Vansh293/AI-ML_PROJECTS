About This Application

This project uses machine learning to estimate insurance costs based on user data. The model was trained on a real-world dataset containing:

  • Age
  • Gender
  • Body Mass Index (BMI)
  • Number of Children
  • Smoking Status
  • Region

We built this to demonstrate how predictive modeling can help analyze and manage healthcare costs.
